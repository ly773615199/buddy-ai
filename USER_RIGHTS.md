# 用户权利指南

> 最后更新：2026-05-10

## 一、您的权利

作为 Buddy 用户，您享有以下数据权利：

### 1.1 知情权

您有权知道 Buddy 采集了什么数据、如何使用、存储在哪里。

**如何行使**：
- 查看隐私面板（设置 → 隐私）
- 阅读隐私政策（PRIVACY_POLICY.md）
- 查看数据处理记录（DATA_PROCESSING_RECORD.md）

### 1.2 访问权

您有权查看 Buddy 持有的所有您的数据。

**如何行使**：
- 打开隐私面板查看所有数据类别
- 使用数据导出功能获取完整数据副本

### 1.3 导出权

您有权以通用格式导出您的所有数据。

**如何行使**：
- 设置 → 隐私 → 导出数据
- API: `GET /api/privacy/export`
- 导出格式：JSON（包含对话、知识、配置、养成数据）

### 1.4 删除权

您有权删除 Buddy 持有的所有您的数据。

**如何行使**：
- 设置 → 隐私 → 删除所有数据
- API: `DELETE /api/privacy/data`
- 删除后不可恢复，请谨慎操作

**删除范围**：
- 所有对话记录
- 所有知识数据
- 所有养成数据（亲密度、进化、行为信号）
- 所有配置数据
- 所有审计日志

### 1.5 撤回同意权

您有权随时撤回对任何数据处理的同意。

**如何行使**：
- 设置 → 隐私 → 关闭对应的传感器/功能
- 关闭摄像头/麦克风/位置
- 关闭第三方服务
- 开启隐私模式（一键关闭所有传感器）

**撤回效果**：即时生效，相关功能立即停止。

### 1.6 限制处理权

您有权限制 Buddy 对您数据的处理。

**如何行使**：
- 开启隐私模式
- 限制特定传感器的使用
- 选择本地模型（避免云端传输）

### 1.7 反对权

您有权反对特定的数据处理活动。

**如何行使**：
- 关闭对应功能
- 联系项目维护者

## 二、数据删除流程

### 2.1 通过界面删除

1. 打开 Buddy 设置
2. 进入「隐私」面板
3. 点击「删除所有数据」
4. 确认删除操作
5. 所有数据立即清除

### 2.2 通过 API 删除

```bash
curl -X DELETE http://localhost:8765/api/privacy/data
```

响应：
```json
{
  "success": true,
  "deleted": {
    "conversations": true,
    "knowledge": true,
    "pet": true,
    "config": true,
    "audit": true
  },
  "timestamp": 1234567890
}
```

### 2.3 删除确认

删除操作不可逆。删除前请确认：
- 已导出需要保留的数据
- 理解删除后无法恢复

## 三、数据导出流程

### 3.1 通过界面导出

1. 打开 Buddy 设置
2. 进入「隐私」面板
3. 点击「导出数据」
4. 选择导出范围
5. 下载 JSON 文件

### 3.2 通过 API 导出

```bash
curl http://localhost:8765/api/privacy/export
```

响应包含：
- 对话历史
- 知识数据
- 养成数据
- 配置数据
- 审计日志

## 四、传感器控制

### 4.1 摄像头

- **开启**：设置 → 感知 → 摄像头
- **关闭**：同上，或开启隐私模式
- **指示**：开启时界面显示红色指示灯
- **数据**：仅内存处理，不存储

### 4.2 麦克风

- **开启**：设置 → 感知 → 麦克风
- **关闭**：同上，或开启隐私模式
- **指示**：开启时界面显示绿色指示灯
- **数据**：识别后即丢

### 4.3 位置

- **开启**：设置 → 感知 → 位置
- **关闭**：同上，或开启隐私模式
- **数据**：粗略位置，本地存储

## 五、隐私模式

隐私模式是最快的隐私保护方式：

- **开启**：设置 → 隐私 → 隐私模式
- **效果**：
  - 禁用所有传感器（摄像头/麦克风/位置）
  - 清除所有临时帧数据
  - 停止所有第三方传输
  - 审计日志记录模式切换
- **关闭**：设置 → 隐私 → 退出隐私模式
