# 数据保护影响评估 (DPIA)

> 最后更新：2026-05-10
> 依据：GDPR Article 35 / EU AI Act

## 一、评估范围

本 DPIA 覆盖 Buddy 应用的所有数据处理活动，重点关注高风险处理：

- 人脸信息处理（生物识别数据）
- 情绪识别（EU AI Act 高风险 AI）
- 音频/视频传感器数据

## 二、数据处理描述

### 2.1 处理活动概述

Buddy 是一个 AI 伴侣应用，具备以下数据处理能力：

1. **对话处理**：用户与 AI 的文本交互
2. **视觉处理**：摄像头画面分析（场景理解、人脸检测、情绪识别）
3. **音频处理**：语音识别和语音交互
4. **位置处理**：环境上下文感知
5. **知识处理**：从对话中提取和积累知识
6. **养成处理**：用户行为分析和个性化

### 2.2 数据类型

| 数据类型 | 敏感度 | 特殊类别 |
|---------|--------|---------|
| 对话文本 | 中 | 否 |
| 摄像头画面 | 高 | 可能包含人脸 |
| 人脸信息 | **极高** | **是（生物识别）** |
| 情绪数据 | 高 | **是（EU AI Act）** |
| 音频片段 | 高 | 可能包含声纹 |
| 位置信息 | 中 | 否 |
| 行为数据 | 低 | 否 |

## 三、风险评估

### 3.1 风险矩阵

| 风险 | 可能性 | 影响 | 风险等级 | 缓解措施 |
|------|--------|------|---------|---------|
| 人脸数据泄露 | 低 | 极高 | **高** | 不持久化、内存处理、5 分钟过期 |
| 情绪识别误用 | 中 | 高 | **高** | 本地处理、用户可关闭、透明度 |
| 音频数据泄露 | 低 | 高 | **中** | 不持久化、识别后即丢 |
| 对话内容泄露 | 中 | 中 | **中** | 本地存储、用户可删除 |
| 第三方传输泄露 | 低 | 中 | **中** | 用户控制、脱敏处理 |
| 位置追踪 | 低 | 中 | **中** | 粗略位置、用户可关闭 |
| 行为画像 | 低 | 低 | **低** | 匿名化、本地处理 |

### 3.2 高风险处理详细评估

#### 人脸信息处理

| 评估项 | 结论 |
|--------|------|
| 必要性 | 场景理解和情绪感知需要人脸检测 |
| 数据最小化 | 仅检测位置，不提取特征向量 |
| 存储 | **不持久化**，仅内存中处理 |
| 过期 | 帧数据 5 分钟自动过期 |
| 用户控制 | 需明确同意，可随时关闭 |
| 替代方案 | 可完全禁用人脸检测 |
| 技术保障 | Canvas 像素化、Shape Detection API |

#### 情绪识别（EU AI Act 高风险）

| 评估项 | 结论 |
|--------|------|
| 透明度 | 告知用户情绪识别的存在和用途 |
| 人类监督 | 用户可随时关闭情绪感知 |
| 准确性 | 作为辅助参考，不作为决策依据 |
| 歧视风险 | 不基于情绪做歧视性判断 |
| 数据使用 | 仅用于个性化交互，不对外传输 |

## 四、缓解措施

### 4.1 技术措施

- **本地处理**：传感器数据仅在内存中处理
- **不持久化**：摄像头/麦克风数据不写入磁盘
- **自动过期**：帧数据 5 分钟自动清理
- **像素化脱敏**：人脸区域自动像素化
- **Shape Detection API**：浏览器原生，无第三方依赖
- **降级链**：检测失败时优雅降级，不阻塞功能

### 4.2 管理措施

- **隐私面板**：用户可查看和管理所有隐私设置
- **审计日志**：记录所有数据操作
- **数据删除**：一键删除所有个人数据
- **数据导出**：用户可导出所有数据
- **隐私模式**：一键禁用所有传感器

### 4.3 用户权利保障

- 知情权：隐私政策 + 隐私面板
- 控制权：随时关闭任何传感器
- 导出权：JSON 格式导出
- 删除权：一键清除
- 撤回权：即时生效

## 五、结论

### 风险等级：可接受（经缓解后）

经过实施上述技术和管理措施后，残余风险降至可接受水平：

- 人脸数据：**低风险**（不持久化 + 用户控制）
- 情绪识别：**低风险**（本地处理 + 透明度 + 人类监督）
- 音频数据：**低风险**（不持久化 + 用户控制）

### 建议

1. 定期审查数据处理实践
2. 更新隐私政策以反映新功能
3. 持续监控 EU AI Act 的实施指南
4. 考虑获取第三方安全审计
