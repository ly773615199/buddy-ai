# Google Play 数据安全声明

> 适用于 Buddy Android 版本

## 数据类型声明

### 数据收集

| 数据类型 | 是否收集 | 用途 | 是否共享 |
|---------|---------|------|---------|
| 应用互动 | 是 | 应用功能 | 否 |
| 已安装的应用 | 否 | - | - |
| 其他操作 | 是 | 应用功能 | 否 |
| 崩溃日志 | 是 | 应用功能 | 否 |
| 诊断 | 是 | 应用功能 | 否 |
| 其他性能数据 | 是 | 应用功能 | 否 |

### 数据传输

| 数据类型 | 是否传输 | 加密 | 可否删除 |
|---------|---------|------|---------|
| 对话文本 | 是（用户配置的 LLM API） | 是 | 是 |
| 音频片段 | 是（用户配置的 STT API） | 是 | 是 |
| 其他数据 | 否 | - | - |

### 设备传感器

| 传感器 | 是否使用 | 用途 | 用户控制 |
|--------|---------|------|---------|
| 摄像头 | 可选 | 场景理解 | 可随时关闭 |
| 麦克风 | 可选 | 语音交互 | 可随时关闭 |
| 位置 | 可选 | 环境感知 | 可随时关闭 |

## 安全实践

- ✅ 数据加密传输
- ✅ 用户可请求删除数据
- ✅ 用户可导出数据
- ✅ 传感器数据不持久化存储
- ✅ 所有第三方传输需用户明确启用
- ✅ 遵循 Families Policy（不针对儿童）

## 填写指南

在 Google Play Console → App content → Data safety 中填写：

1. **数据收集**：选择"是"
2. **数据类型**：选择"应用互动"和"其他操作"
3. **数据用途**：选择"应用功能"
4. **数据共享**：选择"否"（除非用户主动启用云服务）
5. **安全实践**：勾选"数据加密"和"用户可以请求删除数据"
6. **传感器**：声明摄像头、麦克风、位置的使用
